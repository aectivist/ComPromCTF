DESCRIPTION:
ComPromCTF is a tool used for sorting CTF challenges and categories within a user's system. 


HOW TO RUN THE TOOL: 
1. install the shell tool through github
2. install requirements.txt with:
```sed 's/#.*//' requirements.txt | xargs sudo apt-get install -y```

3.run 
```sudo chmod +rwx ComProm.sh```

4. run the tool 
```./ComProm.sh```
